# 📝 Blog Platform

A **Full Stack Blogging Platform** where users can create, read, update, and interact with blog posts.  
The platform provides a clean UI, a modern writing editor, and features like trending blogs, dark mode, and comments.

## 🛠 Tech Stack

- Node.js – Backend runtime
- Express.js – Web framework
- MongoDB Atlas – Cloud database
- EJS – Server-side templating
- Bootstrap / CSS – UI styling
- Render – Deployment platform
- Git & GitHub – Version control

---

## ✨ Features

### 👤 Authentication
- User signup and login
- Secure authentication system

### 📝 Blog Management
- Create blog posts
- Edit existing blogs
- Delete blogs

### 💬 Interaction
- Comment system for blogs
- View count tracking
- Trending blogs based on views

### 🔎 Content Discovery
- Search blogs
- Category filtering
- Pagination support
- Random blog generator

### ✍ Writing Experience
- Notion-style writing editor
- Live preview while writing
- Formatting toolbar (Bold, Headings, Code, Lists, Quotes)
- Reading time estimation

### 🎨 UI/UX
- Modern card-based layout
- Responsive design
- Dark mode support
- Smooth hover animations

## ⚙ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/blog-platform.git
cd blog-platform
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Create Environment Variables

Create a `.env` file:

```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### 4️⃣ Run the Application

```bash
npm start
```

Server will start at:

```
http://localhost:3000
```
